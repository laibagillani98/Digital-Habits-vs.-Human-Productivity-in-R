# Digital-Habits-vs.-Human-Productivity-in-R
Using R to investigate how digital habits such as screen time, social media usage, and notifications impact productivity and focus.

Using a dataset of 6,000 users, I performed data cleaning, transformation, and exploratory data analysis (EDA) in R to uncover behavioral patterns.

## 🎯 Objectives
- Understand relationship between screen time and productivity
- Analyze impact of notifications on focus
- Compare productivity across addiction levels
- Identify patterns in sleep, study, and social media usage

## 🛠 Tools & Technologies
- R (tidyverse, dplyr, tidyr, ggplot2)
- Data Cleaning & Transformation
- Data Visualization
- Statistical Analysis (correlation & regression)
